# Documentation Index

This directory contains project documentation for `meshcore-llm-Lara`.

## Contents

- `configuration.md` — reference for `lara_config.example.yaml`, sections, and recommended values.
- `architecture.md` — overview of the code structure, runtime flow, and major components.
- `testing.md` — how to run tests and what the current unit tests cover.
- `reports/` — archived test reports, analysis notes, and historical outputs.
- `legacy/` — older or archived files preserved for reference.

## How to use this folder

Use these files as the primary documentation source for development and maintenance. The root `README.md` provides a high-level overview and quick start instructions, while `docs/` contains deeper, structured references.
